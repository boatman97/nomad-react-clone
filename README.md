## Promise(js Object)
- javscript `비동기` 객체
  - `비동기`
    - DB 2개를 불러오고 싶어, 근데 첫 번째가 너무 긁어오는데 오래걸려
    - 어떻게 해결해? -> `비동기`
- 시나리오를 잡는 방법 제시(통신 여부)
  - response
    - status : 200(True)
    - ok : True of False

## Await & Async
- `promise` 객체를 더욱더 쉽게 표현하는 방법
- `Async`
  - 함수를 `비동기`로 처리하겠다는 뜻
- `await`
  - `sync`
  - 성공 여부를 결정하지 않음
  - API 처리를 불러와서 value를 기다리는 과정(`sync`)
